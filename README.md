# Online Voting System

##  Description
An Online Voting System developed to conduct secure and efficient digital elections.  
This system allows administrators to create elections and manage candidates, while voters can cast their votes easily and securely.

##  Features
- Admin login system
- Create and manage elections
- Add and manage candidates
- Voter registration
- Secure voting process
- Real-time vote counting
- User-friendly interface

  ## 🛠️ Technologies Used
- Java (OOP Concepts)
- Swing (GUI)
- JDBC (Database Connectivity)
- MySQL / PostgreSQL

## 📂 Project Structure
src/

 ├── service/
 ├── ui/
 ├── model/
 └── main/

## ⚙️ How to Run
1. Clone the repository:
   git clone https://github.com/chaudharysaifullah099-cmd/Online-voting-System.git

3. Configure database (MySQL/PostgreSQL)

4. Run the main class
